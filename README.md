 🎬 CineVista

CineVista es un sitio web desarrollado como parte de una práctica de desarrollo web. El proyecto presenta una cartelera de películas con reseñas, información y recomendaciones de distintos géneros.

Descripción

El objetivo del proyecto es crear un sitio web utilizando **HTML semántico** y **CSS**, aplicando principios de diseño fluido para que el contenido se adapte a diferentes tamaños de pantalla.

El sitio permite explorar diferentes películas, conocer información sobre ellas y navegar entre distintas secciones del sitio.

 Estructura del proyecto

El proyecto incluye las siguientes páginas:

- index.html → página principal
- catalogo.html → lista de películas
- detalle.html → información detallada de una película
- acerca.html → información sobre el sitio
- contacto.html → formulario de contacto
- faq.html → preguntas frecuentes
- tendencias.html → películas populares
- comunidad.html → sección de comunidad

También incluye:

- **css/** → archivos de estilos
- **assets/** → imágenes y recursos visuales

 Tecnologías utilizadas

- HTML5
- CSS3
- Diseño fluido (sin media queries en el layout principal)

 Proyecto académico

Este proyecto fue desarrollado como parte de una práctica académica para el curso de desarrollo web.
